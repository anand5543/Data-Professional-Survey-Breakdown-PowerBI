# Data Professional Survey Breakdown (Interactive Dashboard creation using MS Power BI)
## Project Objective
The goal of this project is to analyze a real-world survey taken by data professionals across the globe. So that, we can understand the current state of the data industry — including salaries, job satisfaction, and barriers to entry — to help aspiring data professionals make informed career decisions.
## Dataset used
- <a href="https://github.com/anand5543/Data-Professional-Survey-Breakdown_PowerBI/blob/main/Power%20BI%20-%20Final%20Project.xlsx">Dataset</a>
## Questions (KPIs)

- How many people took the survey and what is the average age of survey takers?
- What is the average salary by job title?
- Which programming language is the most favourite among data professionals?
- Which countries do the survey takers belong to?
- How happy are data professionals with their Work/Life Balance?
- How happy are data professionals with their Salary?
- How difficult was it to break into the data field?
- Dashboard Interaction <a href="https://github.com/anand5543/Data-Professional-Survey-Breakdown_PowerBI/blob/main/Screenshot%202026-06-05%20142447.png">View Dashboard</a>

## Process

- Verified data for any missing values and anomalies, and sorted out the same.
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created measures and calculated columns using DAX according to the questions asked.
- Merged all visuals into one dashboard and applied slicers to make it dynamic and interactive.
